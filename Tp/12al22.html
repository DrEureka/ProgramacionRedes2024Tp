Compatible con el estándar IEEE 802.11n • IEEE 802.11ax: Establecida en 1019, utiliza frecuencias 2.4GHz y 5GHz. Un proxy es un intermediario que sirve para peticiones de diversos recursos entre un cliente y un servidor 14- Explicar el protocolo Spanning tree. El protocolo Spanning Tree es mayormente conocido por sus siglas STP, dentro de un árbol o red de switches, su función principal es la de organizar esta red para evitar bucles y falta de comunicación de datos. Para esto elige un switch principal (root bridge) que va a tener la prioridad más baja y la dirección MAC más baja, luego cada switch de la red va a poder elegir cual es el mejor puerto para comunicarse directamente o más rápido con el switch principal, y si un puerto de un switch está justo en un camino que podría crear un bucle, el STP lo bloquea temporalmente. 15- Explicar el protocolo de comunicaciones OSPF. Este protocolo (Open Shortest Path First) se utiliza en redes que son grandes ya que este protocolo ayuda a encontrar caminos que son más cortos para que los datos puedan viajar entre routers de una misma red, el OSPF divide la red en áreas y cada router comparte información con otros routers para que todos tengan un mismo mapa de red, así se mantenga una comunicación más rápida y eficiente dentro de la red. 16- Explicar el protocolo ARP. El protocolo ARP (Address Resolution Protocol) es como un "traductor" que convierte las direcciones IP de Internet en direcciones físicas de los dispositivos que están conectados a una red local (LAN). Una explicación "gráfica" que intenta explicar esto es, por ejemplo, tener 2 dispositivos, A y B, que quieren comunicarse, si están en redes diferentes, utilizan direcciones IP para encontrarse, pero si están en la misma red local, necesitan conocer la "dirección física" de cada uno, entonces acá aparece el protocolo ARP que facilita la comunicación entre estos 2 dispositivos. 17- ¿Qué es un Firewall? El Firewall es un sistema de seguridad que utilizan los servidores para protege a tu red de tráfico no deseado 18- ¿Qué es una DMZ? Una DMZ (o zona desmilitarizada) es una red que protege tu red local contra el trafico no deseado. Te permite acceder a redes no confiables, pero a la vez protege a la tuya de éstas. 19- ¿Qué es un Gateway? En palabras simples, un Gateway es un componente que hace como punto de acceso o salida entre dos redes o sistemas que operan utilizando protocolos de comunicación distintos. Entonces, este componente sería como un puente virtual que traduce los datos entre los formatos compatibles con cada una de las redes, facilitando la comunicación y el intercambio de datos. 20- Según Microsoft, ¿qué significa NBL?
